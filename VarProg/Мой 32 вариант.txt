ВАРИАНТ 32: Анализ крови
============================================================
Описание: Результаты гематологических анализаторов: гемоглобин, глюкоза, лейкоциты, тромбоциты. Мониторинг показателей для коррекции терапии.

 Поля:
  • ts (int32): время анализа
  • lab_id (int16): ID лаборатории
  • hgb (float32): гемоглобин, г/л
  • glu (float32): глюкоза, ммоль/л
  • wbc (float32): лейкоциты, 10⁹/л
  • plt (float32): тромбоциты, 10⁹/л

Очистка: glu ∉ [20, 500] → clip. Заменить wbc < 0 на 0. Clip hgb по 95%.
Пояснение: Глюкоза ограничена физиологическими пределами. Лейкоциты не могут быть отрицательными → 0. Гемоглобин ограничьте 95-м процентилем, отсечив редкие патологии.

Группировка: По lab_id. Средний glu, макс plt.
Пояснение: Для каждой лаборатории посчитайте средний уровень сахара и максимальное число тромбоцитов. Это покажет точность калибровки оборудования и популяционные нормы.

Окно: k=30. Moving avg glu. np.diff(hgb) → динамика гемоглобина.
Пояснение: Сглаженная глюкоза выявит стабильность метаболизма. Изменения гемоглобина между анализами укажут на эффективность лечения анемии.

Акцент: Физиологические границы жёсткие, NaN только в glu.
Пояснение: Пропуски допускаются только при сбое анализа глюкозы. Остальные показатели должны быть заполнены. Не заменяйте NaN средними значениями по столбцу.
